# 🔯 JyotiDarshan — জ্যোতির্দর্শন

A complete Vedic Astrology web application built with HTML, CSS & JavaScript.

## Features
- Janma Kundli (Birth Chart)
- Rashi & Nakshatra Calculator
- Vimshottari Dasha
- Panchanga
- Numerology (Pythagorean & Chaldean)
- Kundli Matching (Guna Milan)
- Smart city search with auto lat/lng
- Print & PDF export

## Languages
Bengali • Hindi • English
